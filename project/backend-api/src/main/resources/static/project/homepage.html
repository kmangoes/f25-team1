API Endpoints

Coffee Shops

GET /api/coffeeshops - Get all coffee shops
GET /api/coffeeshops/{id} - Get coffee shop by ID
POST /api/coffeeshops - Create new coffee shop
PUT /api/coffeeshops/{id} - Update coffee shop
DELETE /api/coffeeshops/{id} - Delete coffee shop

Users

POST /api/users/signup - Sign up new user
POST /api/users/login - Login
GET /api/users - Get all users

Events

GET /api/events - Get all events
POST /api/events/user/{userId}/shop/{shopId} - Create event

Reviews

POST /api/reviews/user/{userId}/shop/{shopId} - Create review
GET /api/reviews/shop/{shopId} - Get reviews for a shop

Orders

POST /api/orders/user/{userId} - Create order
GET /api/orders/user/{userId} - Get user orders

Profiles

GET /api/profiles/user/{userId} - Get user profile
PUT /api/profiles/user/{userId} - Update user profile
